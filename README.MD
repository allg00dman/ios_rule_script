# iOS Rule Script (Surge)

> 来自 [blackmatrix7/ios_rule_script](https://github.com/blackmatrix7/ios_rule_script) 的 Surge 分流规则镜像，每日自动同步。

## 说明

- 仅保留 `rule/Surge` 目录，适配 Egern 等代理工具
- 每天 UTC 02:00（北京时间 10:00）自动从上游同步更新
- 规则内容及维护归属原作者 blackmatrix7

## 使用方法

将配置中的规则链接替换为：

```
https://raw.githubusercontent.com/mythos404/ios_rule_script/master/rule/Surge/{规则名}/{规则名}.list
```

例如：

```
https://raw.githubusercontent.com/mythos404/ios_rule_script/master/rule/Surge/Netflix/Netflix.list
```
